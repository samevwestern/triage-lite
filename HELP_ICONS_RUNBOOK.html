⚙️ Settings Menu Runbook isMenuHelpOpen
Header Settings Overlay
Popup Content / Guides
  • 💾
    Export Data Backup handleExportCSV()

    Allows you to back up your local database sandbox at any time. It downloads an Excel-compatible CSV containing all card columns, priority values, time trackers, description parameters, and checklist logs.

  • 🍏
    Apple iCloud Synchronization syncData()

    Connects your active triage boards across all registered iOS and macOS devices. It synchronizes checklist states, focus sessions, priority levels, and custom labels dynamically through Apple CloudKit channels.

  • 🏷️
    Board Label Studio saveLabels()

    Your central classifications workshop. Define customizable classification labels, assign custom hex code styling, or rename category tags to immediately assign priorities and categorize card widgets.

  • ⚡
    Feature Diagnostics triggerHaptic()

    Runs localized client feature sweeps. Instantly tests hardware-level compatibility including Capacitive Haptics, LocalStorage read/writes, Web Audio synthesizers, and system Calendar notification schedulers.

Functional Relevance: Explains exactly what each background setting does. Clarifies how backups work, where lists are synced, and helps verify phone components before setting reminders.

❓ Board Columns Quick Runbook isDashboardHelpOpen
Main Kanban Board
Popup Content / Guides
  • 🎛️
    Board Columns (Lists) lists state

    Your work is split into three lists: TO DO (pending actions), DOING (active focus), and DONE (completed items). The count shows the active cards in each.

  • 📱
    Mobile Column Navigation swipe event listeners

    On mobile, swipe left or right on the screen to slide between columns, or tap the pagination dots at the top of the board to jump directly to a list.

  • 📄
    Card Interaction setSelectedCardForEdit()

    Tap any card's frame to open the Card Details modal. Tap the orange + icon in the header bar of any column to create a brand new task card.

  • 🔄
    Moving Cards handleReorderCard()

    Tap the orange MOVE ▾ button on the card face to immediately open list-shifting options. On desktop, click and drag cards directly to any list column.

Functional Relevance: Serves as the primary navigation guide. Translates column gestures, pagination dots, column-specific cards creation, and list-movers into standard steps.
